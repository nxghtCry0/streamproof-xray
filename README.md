# Streamproof Xray - Minescript
This project is a simple overlay that allows you to see through walls in Minecraft.

### This project is not finished, nor expected to be finished. It is made for a test of Minescript and it's capabilities in real-time.

## Disclaimer
- This tool is intended for use in Singleplayer or Anarchy servers ONLY.
- Using this on multiplayer servers where cheating is prohibited will result in bans.
- The developers do not condone cheating in competitive environments.

## Features
- Simple overlay that allows you to see through walls in Minecraft.
- Streamproof Overlay
- Minescript Scanner Interface
- Configurable FOV
- Configurable Block Colors
- Configurable Scan Radius
- Configurable Block Types
- Configurable Block Colors
- Configurable Block Types
- Configurable Block Colors
- Configurable Block Types

## Usage Prerequisites
- An instance of Minecraft running Minescript in windowed mode.
- Python installed from the Microsoft Store or from the official Python website.
- The Minescript_scanner.py file in .minecraft/minescript folder.

## Usage
- Clone the repository.
- Run the `main.py` file.
- In Minecraft, open the chat and run the command \minescript_scanner.py
- The overlay should now be visible on your screen.

## Configuration
- The configuration file is `config.json`.
- The configuration file can be edited to change the FOV, block colors, scan radius, block types, and block colors.
